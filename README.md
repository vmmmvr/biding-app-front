#### how to run the project

1- git clone the url

2- install packeges

```
npm install
```

3- create `.env` file and add 
```
NEXT_PUBLIC_API_URL=
```
the value of it should be your backend api mostly will look like `http://localhost:BACKEND_PORT`

4- Run the project
```
npm run dev
```

5- to build the project
```
npm run build
```
